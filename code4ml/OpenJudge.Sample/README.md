
### OPEN JUDGE (A SMALL SAMPLE)

Machine learning (ML) code is on the verge of moving beyond the proof-of-concept stage to achieve 
widespread adoption. Open Judge, a meticulously curated dataset, is built upon the foundation of 
Project CodeNet. This project (https://github.com/IBM/Project_CodeNet) itself is a large-scale 
dataset comprising millions of code samples, where each sample represents an intended solution to 
a distinct coding problem.


## Directory Structure

pXXXX
  | info
  | sample

pXXXX -> The problem XXXX
info -> Metadata
sample -> Code samples

## INFO

description.html -> The problem description.
input.txt -> The problem input.
output.txt -> The problem output.

### SAMPLES

paaaalbbscccccc.d

aaaa -> The problem with this sample is the issue which it intends to solve.
bb -> The language description (00: C, 01: C++).
cccccc -> The sample index.
d -> The programming language extension.
